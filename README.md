# gherkin-rust

A pure Rust implementation of the Gherkin (`.feature` file) language for the Cucumber testing framework.

If you want to run Cucumber tests in Rust, try [cucumber-rust](https://github.com/bbqsrc/cucumber-rust)!

## Usage

```toml
[dependencies]
gherkin-rust = "^0.9"
```
